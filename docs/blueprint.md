# **App Name**: CourseCompass

## Core Features:

- APK Retrieval: Retrieve the Agenda+ APK, specifically focusing on extracting relevant data points as requested by the user.
- Assignment Collection: Automatically gather assignment details from D2L Brightspace.
- Quiz Collection: Scrape and compile available quizzes, questions, and due dates.
- Discussion Collection: Aggregate discussion threads, including posts, replies, and associated deadlines.
- Announcement Collection: Collect course announcements and display them to the user in the application.
- Grade Aggregation: Retrieve student grades from D2L and ALEKS and display them within the application.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke a sense of intellect and focus, inspired by academic environments.
- Background color: Very light grayish-blue (#F0F2F9), providing a calm, unobtrusive backdrop for content.
- Accent color: Soft green (#95CD75) to highlight important elements such as due dates and grades.
- Body and headline font: 'Inter', a sans-serif font that provides a clean and readable user interface.
- Use the provided 'A+' image as the main application icon.
- Employ a card-based layout for organizing information, with clear hierarchy and spacing to avoid clutter.
- Use subtle transition animations when loading new information or navigating between sections to enhance user experience.
